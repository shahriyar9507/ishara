# Design References

These are the visual references the UI theme is based on. **We follow the theme/mood/components,
not the content** — Ishara content is Bangla Sign Language → text & voice. See `../DESIGN.md` for the
full mapping and design tokens.

Drop the two reference PNGs here (filenames expected by docs):

- `aurora-dark.png` — **Reference 1 (dark):** deep navy/midnight background with aurora glow,
  glassmorphism cards, electric-blue glowing energy orb, blue "I'm Listening" waveform, circular glass
  controls + central mic FAB, "Change Language" list screen, "Select Model & Tool" card grid.
  → basis for the **Aurora (dark)** default theme.

- `lavender-light.png` — **Reference 2 (light):** soft lavender→white gradient, frosted glass,
  purple glowing orb, concentric pulse rings around the mic, soft circular buttons, dark text
  ("Voice Recorder", "AI Voice Generator", "Demo: Voice Recording").
  → basis for the **Lavender (light)** theme.
